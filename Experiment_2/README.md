# Aim - To develop an Android application to draw graphical primitives such as line, rectangle, circle, oval, and text using Canvas class.
# Algorthim
  1. Start Android Studio.
  2. Create a new Android project.
  3. Create a custom View class.
  4. Override the onDraw() method.
  5. Create a Paint object to define drawing style.
  6. Use Canvas methods to draw shapes like line, rectangle, circle, and oval.
  7. Display text using drawText() method.
  8. Set the custom view in MainActivity.
  9. Run the application.
  10. Verify that shapes and text are displayed.
