# Experiment 1
# Aim - To develop a simple Android application that displays "Hello World" using a TextView control.
# Algorithm
 1. Start Android Studio.
 2. Create a new Android project using Empty Activity.
 3. Open activity_main.xml file.
 4. Add a TextView component in the layout.
 5. Set the text property to "Hello World".
 6. Open MainActivity.java file.
 7. Set the layout using setContentView().
 8. Run the application on an emulator or Android device.
 9. Verify that "Hello World" is displayed on the screen.
 10. Stop the application.
