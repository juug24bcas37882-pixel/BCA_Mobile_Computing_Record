# Aim - To develop an Android application that displays a slideshow of images using ImageView and Handler.
# Algorithm
  1. Start Android Studio.
  2. Create a new Android project.
  3. Add ImageView in activity_main.xml layout.
  4. Place images in drawable folder.
  5. Declare an array to store image resources.
  6. Use Handler to change images periodically.
  7. Update ImageView with new image every few seconds.
  8. Run the application.
  9. Verify that images change automatically.
  10. Stop the application.
