# Aim - To develop an Android application to perform addition of two numbers using EditText and Button controls.
# Algorithm
  1. Start Android Studio.
  2. Create a new Android project.
  3. Design the layout using activity_main.xml.
  4. Add two EditText fields for input numbers.
  5. Add Add and Reset buttons.
  6. Add a TextView to display the result.
  7. Write Java code to read numbers from EditText.
  8. Perform addition when Add button is clicked.
  9. Display result in TextView.
  10. Run the application and verify output.
